
# 🍽️ Food Menu App

This is a simple Food Menu web application built using React JS.

---

## 🚀 Features

- 🔍 Search food items  
- 📂 Filter by categories (Breakfast, Biryani, Curry, Drinks, etc.)  
- 🖼️ Food images for each item  
- 🛒 Add to cart functionality  
- ⭐ Rating system  
- 🎨 Zomato-style UI design  

---

## 🛠️ Technologies Used

- React JS  
- JavaScript  
- CSS  
- HTML  

---

## 📸 Project Preview

A responsive food ordering UI with search, filters, and cart system.

---

## ▶️ How to Run

1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`  
4. Open in browser at `http://localhost:3000`
